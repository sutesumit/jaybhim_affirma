# Simplify Telegram Notification Code

## Simplification Opportunities Found

### 1. Duplicated IP extraction â€” 9 copies across 7 files [STRUCTURAL]

The same 4-line block is copy-pasted identically in every route:
```typescript
const serverIp = request.headers.get("x-forwarded-for") ?? undefined;
const parsedIp = serverIp?.split(",")[0]?.trim();
const isLocalhost = parsedIp?.startsWith("127.") || parsedIp === "::ffff:127.0.0.1" || parsedIp === "::1";
const ip = isLocalhost ? null : (parsedIp ?? null);
```

**Locations:** likes/route.ts, comments/route.ts, comments/[id]/route.ts (Ă—2), father-son-stories/route.ts, father-son-stories/[storyId]/route.ts (Ă—2), auth/verify-otp/route.ts, auth/logout/route.ts

**Fix:** Extract to `src/lib/notifications/helpers.ts`:
```typescript
export function extractRequestContext(request: Request, user?: { phone?: string | null; email?: string | null } | null) {
  const serverIp = request.headers.get("x-forwarded-for") ?? undefined;
  const parsedIp = serverIp?.split(",")[0]?.trim();
  const isLocalhost = parsedIp?.startsWith("127.") || parsedIp === "::ffff:127.0.0.1" || parsedIp === "::1";
  return {
    ip: isLocalhost ? null : (parsedIp ?? null),
    contact: user?.phone || user?.email || null,
    userName: user?.display_name ?? "Anonymous",
  };
}
```

This turns every route's 7-line block into a 1-liner:
```typescript
const { ip, contact, userName } = extractRequestContext(request, user);
```

**Why this is safe:** The extracted function has zero dependencies â€” pure input â†’ output. No behavior change.

### 2. Duplicated guard in TelegramBotNotifier â€” 10 copies [STRUCTURAL]

Every method in `TelegramBotNotifier` has the same 2-line guard:
```typescript
const ownerChatId = getOwnerChatId();
if (!ownerChatId) return;
```

**Fix:** Extract to a private method:
```typescript
private async send(formatFn: () => string): Promise<void> {
  const ownerChatId = getOwnerChatId();
  if (!ownerChatId) return;
  await sendMessage(ownerChatId, formatFn());
}
```

Each method becomes a one-liner:
```typescript
async notifyLike(like: LikeNotificationPayload): Promise<void> {
  await this.send(() => formatLikeNotification(like));
}
```

**Why this is safe:** The guard logic is identical in all 10 methods. No behavioral difference.
