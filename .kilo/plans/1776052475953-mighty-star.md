## Overview
This plan outlines implementation of a complete SEO metadata system including:
- Centralized metadata configuration
- HTML metadata builders (OpenGraph, Twitter, canonical URLs)
- JSON-LD Structured Data (Schema.org)
- Dynamic OG Image generation
- XML Sitemap with proper priorities and change frequencies
- RSS Feed

## Design Decisions Adopted from Blueprint
1. ✅ Single source of truth configuration file
2. ✅ Explicit type safety with `as const satisfies` patterns
3. ✅ Separate builder functions for static/dynamic pages
4. ✅ Custom XML sitemap route for full control
5. ✅ Declarative `imagePolicy` for OG images
6. ✅ Unified canonical URL builder
7. ✅ Edge-cached dynamic OG image routes

## File Structure
```
src/
├── config/
│   └── metadata.ts               <- Site constants & static page registry
├── lib/

│       ├── builders.ts           <- Metadata construction functions

│       └── og-image.tsx          <- OG image generation utilities

    └── og/
        ├── static/[page]/
        │   └── route.tsx         <- Static OG image routes
        └── [content]/[slug]/
            └── route.tsx         <- Dynamic OG image routes
```
